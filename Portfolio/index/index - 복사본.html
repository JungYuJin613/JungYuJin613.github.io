About Me

웹을 즐기는 웹퍼블리셔 정유진입니다.
저는 좋은 웹사이트(Good Wed)를 만드는 것이 저의 목표입니다.
이를 위해 웹 퍼블리셔로서 웹 디자이너로서의 안목을 높이기 위해 항상 공부하는 자세로
수 많은 웹 사이트를 접하고 코드를 분석하며 제 것으로 만드는 것을 즐깁니다.
좋은 웹 사이트란 웹 사이트의 목적에 맞는 필요한 요소만을 재료로 사용하여 직관적이며 빠르고,
모든 이들이 이용할 수 있는 것이라고 생각합니다.
좋은 웹을 만들기위해 항상 제자리에 있지 않고 발전해 나가는
웹 퍼블리셔가 되기 위해 노력하고 있습니다.

SKILLS

HTML5

HTML5 등 DOCTYPE을 이해하고 문서에 맞춰 작업이 가능합니다. 의미에 맞는 태그를 사용하여 구조가 잘 짜인 마크업을 할 수 있습니다.

CSS3

그래픽 효과를 구현할 수 있는 CSS3는 디자인을 통해 이미지로 고정되는 기능을 웹 브라우저에 지원합니다.
text-shadow,opacity,box-shadow, border-radius, transition 등을 사용할 줄 압니다.

JAVASCRIPT

절차지향/객체지향을 이해하며 함수, 메소드, 클래스, 프로토타입 등 다양하게 활용합니다. 그리고 let, const, Arrow Function과 같은 ES6 기초를 이해합니다.

JQUERY

다양한 메소드를 사용해 기능을 구현합니다.
또한 항상 웹속도와 효율을 고려하며 자바스크립트와 함께 활용하고 있습니다. 플러그인 API 활용, 수정이 가능합니다.

Mobile/반응형웹

UI/UX를 생각하며 다양한 기기, 브라우저에 최적화된
퍼블리싱을 합니다. 유동적으로 변화하는 반응형웹도
제작이 가능하며 Bootstrap 경험이 있습니다.

PHOTOSHOP

간단한 이미지 보정과 편집을 할 수 있습니다. 플랫 디자인과 레이아웃 UX/UI 디자인의 요소와 컨셉을 제작 할 수 있습니다.
